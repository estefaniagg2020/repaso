Grid repeat

Permite indicar cuántas filas o columnas quieres con el mismo valor

            grid-template-columns:repeat(4,1fr);
            grid-template-rows:repeat(4,1fr);

Hará 4 columnas de una fracción cada una y 4 filas

Justify-content

Alineacion horizontal con respecto al contendor

Align-items/content

Alineacion vertical con respecto al contendor

Para alinear los elementos de dentro es mejor ponerle a cada elemento display grid, align-content center.

Espacio a ocupar

grid-columns: 2/4 le indicamos que empiece en la columna dos y acabe en la 4. Es igual a 

            grid-columns-start:2;
            grid-columns-end:4;

Align-self

Se le pone a los objetos, y permite alinearnos verticalmente
